# Dogs Bootstrap Project

A responsive Bootstrap-based dog breeds showcase page featuring breed cards, images, navigation, and a clean grid layout.

## Features
- Responsive Bootstrap navbar with collapsible mobile menu.
- Dog breed cards with images, descriptions, and action buttons.
- Two-row responsive card grid.
- Bootstrap utilities for spacing, shadows, containers, and buttons.
- Simple footer section.

## Tech Stack
HTML, Bootstrap 5
How to run:
Open index.html in browser.
